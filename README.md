# TimeFácil App

Aplicativo profissional para sorteio e escalação de times de futsal.

Desenvolvedor: **BlackCrisper**

## Tecnologias

- Vite
- React + TypeScript
- Tailwind CSS + shadcn/ui
- PWA (vite-plugin-pwa)

## Como rodar localmente

```sh
npm install
npm run dev
```

## Build de produção

```sh
npm run build
npm run preview
```
